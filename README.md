# 🗾 おでかけプランナー

富山県内のおでかけスポットを条件検索で提案するWebアプリです。

## 🌐 公開URL

https://odesuke-planner.onrender.com

## 📱 アプリ概要

- **対象エリア**：富山県全域
- **ターゲット**：子連れファミリー・カップル・友人同士・一人旅
- **目的**：条件に合う富山県内のおでかけスポットを提案する

## ✨ 機能

- エリア・同行者・ジャンル・所要時間の4条件でスポット検索
- 検索結果の一覧表示
- スポット詳細ページ
- スマホ対応（レスポンシブデザイン）

## 🛠 使用技術

| カテゴリ | 技術 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript |
| バックエンド | Python / Flask |
| データベース | SQLite |
| ソース管理 | GitHub |
